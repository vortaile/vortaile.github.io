# Vortaile's Blog

A personal blog and knowledge base where I document whatever I find interesting.

**Live site:** [https://vortaile.github.io](https://vortaile.github.io)

## About

This blog is my public notebook. I write about things I build, things I break, and things I learn along the way. No specific niche, just whatever feels worth writing about:

- **Project writeups** and documentation
- **CTF writeups** and security research
- **Pentesting**, exploit development, and PoC
- **DFIR**, forensics, and incident response
- **Technical deep-dives** on tools, concepts, or random rabbit holes
- Anything else I feel like putting into words

All posts are written in Markdown and organized by categories and tags so you can find what interests you.
